Behavior

  • "Set as Default Schema" — appears at the bottom of the dropdown, separated by a divider. Applies the currently selected schema as the default for this connection.
  • ★ default indicator — the default schema shows a subtle badge so the user knows which is default.
  • Clearing — selecting "All Schemas" + "Set as Default Schema" clears the default (reverts to no default).
  • New tabs — new tabs on this connection automatically start with the default schema selected.
