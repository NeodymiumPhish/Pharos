## Default Shortcuts

### Query Operations

| Action | Shortcut | Description |
|:-------|:---------|:------------|
| Run Query | `Cmd+Enter` | Execute the current query |
| Save Query | `Cmd+S` | Save current query to library |
| Cancel Query | `Escape` | Cancel the running query |
| Export CSV | `Cmd+E` | Export results as CSV |
| Copy Results | `Cmd+Shift+C` | Copy results to clipboard |

### Tab Management

| Action | Shortcut | Description |
|:-------|:---------|:------------|
| New Tab | `Cmd+T` | Create a new query tab |
| Close Tab | `Cmd+W` | Close the current tab |
| Reopen Tab | `Cmd+Shift+T` | Reopen the last closed tab |
| Next Tab | `Cmd+]` | Switch to the next tab |
| Previous Tab | `Cmd+[` | Switch to the previous tab |
| Tab 1–8 | `Cmd+1` – `Cmd+8` | Switch to a specific tab |
| Last Tab | `Cmd+9` | Switch to the last tab |

### Editor

| Action | Shortcut | Description |
|:-------|:---------|:------------|
| Format SQL | `Shift+Alt+F` | Auto-format the SQL in the editor |
| Find in Results | `Cmd+F` | Open the find bar in the results grid |

### Inline Editing

| Action | Shortcut | Description |
|:-------|:---------|:------------|
| Delete Row | `Delete` or `Backspace` | Mark selected rows for deletion |
| Confirm Edit | `Enter` or `Tab` | Confirm the current cell edit |
| Cancel Edit | `Escape` | Cancel the current cell edit |

## Customizing Shortcuts

1. Open **Settings** > **Keyboard**
2. Find the shortcut you want to change
3. Click the shortcut field to enter recording mode
4. Press the new key combination
5. The shortcut updates immediately

All customizable shortcuts support the following modifiers:
- `Cmd` (Command)
- `Shift`
- `Alt` (Option)

Modifiers can be combined (e.g., `Cmd+Shift+S`).

## Resetting Shortcuts

Click **Reset Shortcuts** in the Keyboard settings tab to restore all shortcuts to their default values.
